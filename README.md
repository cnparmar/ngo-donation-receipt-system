# 🌍 Free NGO Donation Receipt System

A completely free, no-code system for NGOs to:
- Collect donor information
- Generate professional PDF receipts
- Send receipts automatically via email
- Track donations with reports and dashboard

---

## ✨ Features

- ✅ Automatic PDF receipt generation
- ✅ Email receipts to donors
- ✅ Sequential receipt numbering (audit-safe)
- ✅ Multi-currency support (INR / USD)
- ✅ Google Sheets-based reporting
- ✅ Zero cost setup

---

## 🧰 Tools Used

- Google Forms
- Google Sheets
- Google Docs
- Google Apps Script

---

## ⏱ Setup Time

30–60 minutes

---

## 👥 Who is this for?

- Small NGOs
- Charities
- Non-profits
- Community organizations

---

## 🚀 Quick Start

1. Create Google Form
2. Link to Google Sheet
3. Create receipt template in Google Docs
4. Paste script from `/scripts/apps-script.js`
5. Add trigger (on form submit)

👉 Detailed steps: See `SETUP_GUIDE.md`

---

## ⚠️ Disclaimer

This is a basic system. For legal/tax compliance (like 80G in India or 501(c)(3) in US), ensure your receipt format meets regulatory requirements.
